# Documentation

This directory is for public, repository-shipped documentation.

## Scope

- User-facing and contributor-facing material that is appropriate for the public repository.
- Stable references that explain how to use, build, or validate the project.

## Exclusions

- Internal implementation logs and agent-operational notes.
- Do not keep those logs in the repository unless explicitly requested.

## Primary Entry Points

- `README.md` (project overview and usage)
- `CONTRIBUTING.md` (contributor workflow)
